## 🚀 Quick Start

Clone the repository:

```bash
git clone https://github.com/oymusafir/myportfolio-nextjs.git
````

Navigate into the folder:

```bash
cd myportfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run start
```
